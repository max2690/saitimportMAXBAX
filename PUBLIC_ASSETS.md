# Медиафайлы в `public/`

Если при клонировании с GitHub не всё загрузилось (тяжёлые файлы иногда не комитят), положите недостающие файлы вручную.

## Обязательно для херо

| Путь | Назначение |
|------|------------|
| `public/video/hero.mp4` | Фоновое видео в блоке «Импорт из Китая под ключ». Если файла нет или он пустой — показывается градиентный фон. |
| `public/hero/hero-main-1920.jpg` | Первый кадр / постер: показывается пока видео грузится и **если видео не загрузилось** — остаётся этой картинкой. |

## Виды доставки (блок с ЖД / Авиа / Авто / Море)

| Путь | Назначение |
|------|------------|
| `public/video/rail.mp4` | Видео для «ЖД» |
| `public/video/air.mp4` | Видео для «Авиа» |
| `public/video/road.mp4` | Видео для «Авто» |
| `public/video/sea.mp4` | Видео для «Море» |

## Остальное

| Путь | Назначение |
|------|------------|
| `public/logo.svg` | Логотип (уже в репе). |
| `public/assets/` | Папки для картинок кейсов, команды и т.д. — при необходимости добавьте изображения; в коде пока используются только текстовые блоки. |

После добавления `public/video/hero.mp4` перезагрузите страницу — херо подхватит видео автоматически.
